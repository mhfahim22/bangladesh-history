# Heritage of Bengal — বাংলাদেশের ঐতিহাসিক স্থান

A static website showcasing 8 historical places of Bangladesh, generated by the **Aurora** programming language (Static Site Generator pattern).

## Languages & Technologies Used

| Language/Tech | Purpose |
|--------------|---------|
| **Aurora** | Static Site Generator — compiles templates & writes `.html` files |
| **HTML** | 12 page templates (index, places, 8 detail pages, about, base) |
| **CSS** | Inline styles with custom design system (Playfair Display + Inter fonts) |
| **GitHub Actions** | CI/CD — builds site on `git push`, deploys to GitHub Pages |
| **GitHub Pages** | Free static hosting |

## Architecture

- **`build.aura`** — Aurora SSG: compiles templates, renders HTML, writes to `output/`
- **`server.aura`** — Aurora backend server (alternative: run locally on port 8080)
- **`templates/`** — 12 self-contained HTML files with inline CSS
- **`.github/workflows/deploy.yml`** — GitHub Actions: auto-build & deploy

## How to Run Locally

### Static build (SSG):
```bash
bin\aurorac.exe build.aura
```
Generated files in `output/` — open any `.html` in browser.

### Live server:
```bash
aurorac.exe server.aura
```
Server starts at `http://localhost:8080`

## Auto-Deploy

Every push to `master` triggers GitHub Actions:
1. `aurorac.exe build.aura` → generates `output/`
2. Deploys `output/` to GitHub Pages
